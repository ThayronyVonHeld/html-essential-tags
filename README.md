> **Este projeto é uma documentação prática dos meus estudos em HTML**

Este projeto é um guia visual e prático produzido durante meus estudos de HTML. A proposta é organizar, de forma clara e progressiva, as principais tags e conceitos fundamentais da linguagem. Tudo organizado de forma didática para consulta rápida e reforço do aprendizado.

## 🌐 Acesse o projeto
🔗 https://thayronyvonheld.github.io/html-essential-tags/

---
## 🎯 Objetivo

Este projeto não tem como foco ser um produto final, mas sim um **material em constante evolução**.
A ideia é:

- 📖 Material de consulta rápida
- 📈 Consolidar os fundamentos do HTML
- 💡 Exemplos práticos para estudo
- 🚀 Criar uma base sólida para estudos futuros em CSS e JavaScript
